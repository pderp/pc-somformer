# PC-SOMFormer

**Pattern Completion Self-Organizing Map (PC-SOMFormer)**

Exploratory work on integrating Probabilistic Logic Networks (PLN) and Economic Attention Networks (ECAN) with Self-Organizing Map (SOM) architectures for pattern completion and associative memory.

## Goals

- Explore how SOM topologies can be used for pattern storage and completion
- Integrate PLN inference with SOM-based memory structures
- Investigate ECAN attention allocation over SOM neighborhoods
- Document findings and prototypes transparently

## Status

Early-stage research and prototyping. Documentation in progress.
